Game Instructions

How to Play:

Touch the lit bombs before they explode.

The bomb will slowly start to change colour until it explodes




The game will end when the bomb has not been defused and explodes.

Game Modes:

Original Game

See how many bombs you can defuse and try and set a high score

Custom Game

In custom game mode you will be able to change the fuse length and the max number of live bombs. You can also increase and decrease the grid size.
